# 安全政策

## 支持的版本

目前仅维护最新版本的安全更新。建议及时升级到最新版本。

## 报告安全漏洞

如果您发现安全漏洞，**请不要**公开发布 Issue 或 Pull Request。

### 报告流程

1. 发送邮件至：daofengqianlin@silverdragon.cn
2. 邮件标题格式：`[安全漏洞] 简要描述`
3. 请提供以下信息：
   - 漏洞描述
   - 影响版本
   - 复现步骤
   - 潜在影响
   -（可选）修复建议

### 响应承诺

- 我们将在 **48 小时内**确认收到报告
- 在 **7 天内**评估漏洞严重性并制定修复计划
- 修复完成后将发布安全公告

### 安全最佳实践

部署 Draurls 时，请确保：

- 修改所有默认密码和密钥
- 使用强随机字符串作为 `JWT_SECRET`
- 启用 HTTPS（生产环境）
- 限制数据库和 Redis 的外部访问
- 定期更新依赖包
- 配置适当的防火墙规则
- 使用环境变量存储敏感配置

## 已知安全问题

当前没有已知的未修复安全漏洞。
